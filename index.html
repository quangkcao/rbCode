rbCode
jQuery plugin

Change the way of sharing codes in pre tag.

    <pre class="code>
    With rbCode, code lines is different.
    Click buttons at right-top corner to see effects
    </pre>

Features

  • Box code lines
  • Zebra effect on code lines
  • Numberizes those code lines
  • Popout to the center of the window

Options

Parameter Type Default Value Description
is_number Boolean false indicate show number or not
is_zebra Boolean false indicate show zebra effect or not
zebra_bg String #CCC background of zebra effect
zebra_cross_bg String #FFF background of zebra cross lines
modal_bg String #000 background of modal window
bg String #CCC background of box
text_color String #000 text color
list_color String #CCC color of list style such number
z_index Integer 999 popout 999 and modal window 999-1
opacity Number 0.75 transperancy of modal window
a_number Array ['Hide Number','Show Number','text'] label Number button. Default type: 'text'. If wanted image button, change to [url,url,'image']
a_zebra Array ['Hide Zebra','Show Zebra','text'] label Zebra button. Default type: 'text'. If wanted image button, change to [url,url,'image']
a_popout Array ['Pop Out','Close','text'] label Popout button. Default type: 'text'. If wanted image button, change to [url,url,'image']

Custom Style

    <pre class="code>
    You can change it to different color or modal window color by edit parameters.
    </pre>
    <script type="text/javascript">
    $('pre.code').rbCode({zebra_bg:'#D4E5E5',
                        zebra_cross_bg:'#FFF',
                        modal_bg:'#D4E5E5',
                        bg:'#D4E5E5',
                        opacity: 0.5,
                        a_number:['http://runbusi.com/upload/article/number.png',
                                    'http://runbusi.com/upload/article/number.png',
                                'image'
                                ],
                        a_zebra:['http://runbusi.com/upload/article/zebra.png',
                                'http://runbusi.com/upload/article/zebra.png',
                                'image'
                                ],
                        a_popout:['http://runbusi.com/upload/article/popout.png',
                                    'http://runbusi.com/upload/article/close.png',
                                'image'
                                ],
                        text_color:'#336699'
                        })
    </script>
